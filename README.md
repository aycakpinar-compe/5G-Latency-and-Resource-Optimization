# 5G-Latency-and-Resource-Optimization
Machine learning project for predicting network latency and optimizing resource allocation in 5G networks.
About This Project
In this project, I worked on a 5G network dataset to understand how different factors like signal strength and bandwidth affect network performance.
My main goal was to analyze the data and build a simple model to predict latency.

What I Did so far :
Loaded and explored the dataset using Python
Cleaned the data (removed units like ms, dBm, %, Mbps, Kbps)
Converted all values into numeric format
Created a new feature called efficiency as (allocated / required bandwidth)
